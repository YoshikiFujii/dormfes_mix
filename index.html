大抽選会のゲームルール

会場で実施する各ゲームの進め方と勝敗条件を、PDFのレイアウトに近い紙面風デザインでまとめました。

目次
あっち向いてホイ 押し相撲 早押しクイズ 高速しりとり 風船爆弾ゲーム

GAME 1 あっち向いてホイ

  1. 初めに「じゃんけん」をします。
  2. 勝った人が「指を指す役」、負けた人が「顔を向ける役」になります。
  3. 勝った人は「あっち向むいてホイ」と言いながら、上下左右のいずれかを指差します。
  4. 負けた人はそのタイミングで、指された方向とは違う方向に顔を向ければセーフ。
  5. 指した方向と顔を向けた方向が同じだったら、指した人の勝利。

GAME 2 押し相撲

  1. 位置決め：2 人が向かい合い、足は揃えて立ちます。
  2. 合図：行司役の人の「はっけよい、残った！」の合図で始めます。
  3. 押し合い：お互いに手のひらで押し合います。
  4. 勝敗：バランスを崩して足が動いたり、地面から離れた方が負けです。
※体を直接掴んだり、相手の体を叩いたりしてはいけません。
※相手の足が動いたり、浮いたりしたら負けとなります。

GAME 3 早押しクイズ

  1. 出題：司会（出題者）が問題文を読み始めます。
  2. 解答権の獲得：参加者は「分かった！」と思った時点でボタンを押し解答権を得ます。
    ※早く押した人が優先です。
  3. 解答：押した人が答えます。
  4. 正誤判定：
    • 正解 → 得点（1 点）
    • 不正解 → 解答権を失い、相手に解答権が移る。
  5. 勝敗の決定：2 点を先に取った方が勝利。

GAME 4 高速しりとり

  1. 基本：前の人が言った言葉の最後の文字で始まる言葉をつなげていきます。
  2. テンポ：回答のテンポは手で「パンパン」と 2 回叩くスピード（司会が例を見せます）。
  3. 禁止事項：
    • 「ん」で終わる言葉を言ったら負け。
    • 単語（名詞）以外の文章等はなし。
    • 語尾が伸ばし棒の単語の場合は一つ前の言葉（例：ルビー→「ビ」）。
    • 語尾が濁点の言葉は濁点なしに変更不可（例：うさぎ→「ぎ」のみ、「き」は不可）。
    • 語尾が小文字の場合は小文字／前の文字どちらでも可（例：キティ→「イ」でも「ティ」でも可）。
  4. 勝敗：脱落して最後まで残った人が勝利。

GAME 5 風船爆弾ゲーム

  1. 順番：
    • 参加者が交代で風船に空気入れで空気を入れます。
    • 1 回に 1 回以上ポンプを押す。
    • 1 回以上なら何回押しても良い。
  2. 勝敗：
    • 自分の番で風船を割ってしまった人が負け。
    • 割らずに回避できた人はセーフ。
    • このゲームは 1〜3 等でのみ実施。
